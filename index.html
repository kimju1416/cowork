개요
Claude Cowork란?
옆자리에 앉아 내 컴퓨터로 직접 일해주는 AI 동료
내 PC에서 직접 작업

바탕화면·문서 폴더의 파일을 직접 읽고, 새 파일을 만들어 저장합니다. 업로드·다운로드가 필요 없습니다.

여러 단계를 알아서

"읽고 → 분석하고 → 정리해서 → 저장"을 한 번 시키면 순서대로 끝까지 처리합니다.

앱·인터넷과 연결

구글 드라이브·캘린더·Gmail, 웹 검색까지 연결해 실제 데이터로 작업합니다.

예약 자동 실행

매일 아침, 매주 월요일처럼 정해진 시간에 같은 작업을 자동 반복할 수 있습니다.

NEW · 원격 세션 예약이라면
클라우드에서 실행되므로 PC가 꺼져 있어도 정시에 실행됩니다 (베타 · Max 플랜부터 순차 확대). 단, 내 PC 폴더·카카오톡 같은 로컬 연동을 쓰는 작업은 데스크톱 앱이 열려 있어야 해요 → 다음 장 참고

hourly / daily / weekly / on weekdays / manually
